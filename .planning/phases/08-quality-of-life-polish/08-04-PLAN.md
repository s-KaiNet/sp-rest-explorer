**UAT Gap 3:** The header, breadcrumb bar, and sidebar are too dark in dark mode because they use the same background as the page (`bg-background` at oklch 0.14) or darker (`--sidebar-background` at oklch 0.12). GitHub Dark uses an elevation pattern where chrome surfaces are LIGHTER than the page background (#161b22 ≈ oklch 0.18 vs #0d1117 ≈ oklch 0.14).

**index.css — Raise sidebar-background:**
In the `.dark` block, change `--sidebar-background` from `oklch(0.12 0.005 260)` to `oklch(0.18 0.005 260)`. This aligns with GitHub Dark's #161b22 for elevated surfaces. Also change `--sidebar-primary-foreground` from `oklch(0.12 0.005 260)` to `oklch(0.18 0.005 260)` to keep the pair consistent.

Specifically on line 74: `--sidebar-background: oklch(0.12 0.005 260);` → `--sidebar-background: oklch(0.18 0.005 260);`
And line 77: `--sidebar-primary-foreground: oklch(0.12 0.005 260);` → `--sidebar-primary-foreground: oklch(0.18 0.005 260);`

**Header.tsx — Use elevated surface:**
Line 36: Change `bg-background` to `bg-sidebar` in the `<header>` className. The class currently reads:
`"fixed top-0 z-50 flex h-14 w-full items-center border-b border-border bg-background"`
Change to:
`"fixed top-0 z-50 flex h-14 w-full items-center border-b border-border bg-sidebar"`

This works because in light mode, `--sidebar-background` (oklch 0.985) is nearly identical to `--background` (oklch 1.0) — imperceptible difference. In dark mode, it will be the elevated 0.18 instead of 0.14.

**BreadcrumbBar.tsx — Use elevated surface:**
Line 25: Change `bg-background` to `bg-sidebar` in the `<nav>` className. Currently:
`"group z-10 flex min-h-[40px] shrink-0 flex-wrap items-center border-b border-border bg-background px-4 py-2"`
Change to:
`"group z-10 flex min-h-[40px] shrink-0 flex-wrap items-center border-b border-border bg-sidebar px-4 py-2"`

**ResizablePanel.tsx — Add sidebar background:**
Line 63: The outer div currently has no background class:
`className={`relative shrink-0 border-r border-border ${className ?? ''}`}`
Add `bg-sidebar` to it:
`className={`relative shrink-0 border-r border-border bg-sidebar ${className ?? ''}`}`
