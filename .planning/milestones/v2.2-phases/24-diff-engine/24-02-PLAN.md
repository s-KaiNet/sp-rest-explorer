  <name>Task 1: Create diff-store with useSyncExternalStore pattern</name>
  <files>
    app/src/lib/diff/diff-store.ts
  </files>
  <action>
1. Create `app/src/lib/diff/diff-store.ts` following the EXACT same singleton + useSyncExternalStore pattern as `app/src/lib/metadata/metadata-store.ts`:

   **Module-level state:**
   ```
   let diffResult: DiffChanges | null = null
   let diffStatus: DiffStatus = 'idle'
   let diffError: string | null = null
   const listeners = new Set<() => void>()
   ```

   **Subscription (useSyncExternalStore contract):**
   - `subscribe(callback)` — adds to listeners set, returns unsubscribe function
   - `getResultSnapshot()` — returns `diffResult`
   - `getStatusSnapshot()` — returns `diffStatus`
   - `getErrorSnapshot()` — returns `diffError`

   **React hooks:**
   - `useDiffSnapshot(): DiffChanges | null` — `useSyncExternalStore(subscribe, getResultSnapshot, getResultSnapshot)`
   - `useDiffStatus(): DiffStatus` — `useSyncExternalStore(subscribe, getStatusSnapshot, getStatusSnapshot)`
   - `useDiffError(): string | null` — `useSyncExternalStore(subscribe, getErrorSnapshot, getErrorSnapshot)`

   **Non-React getters (for imperative code):**
   - `getDiffResult(): DiffChanges | null`
   - `getDiffStatus(): DiffStatus`
   - `getDiffError(): string | null`

   **Internal notify function:**
   - `notify()` — calls all listeners (same as metadata-store pattern)

   **Internal set functions:**
   - `setLoading()` — sets status='loading', error=null, notify()
   - `setReady(result: DiffChanges)` — sets diffResult=result, status='ready', error=null, notify()
   - `setError(message: string)` — sets status='error', error=message, diffResult=null, notify()

2. **Orchestration function** `computeDiff(year: number, month: number): Promise<void>`:
   - Sets loading state
   - Gets current metadata via `getMetadata()` from metadata-store. If null, setError("Metadata not loaded yet") and return.
   - Calls `fetchHistoricalBlob(year, month)`. If returns null (404): call setReady with empty DiffChanges `{ entities: [], functions: [] }` and return (graceful empty state — per CONTEXT.md decision).
   - Calls `computeRawDiff(currentMetadata, historicalMetadata)`
   - If delta is undefined (identical): setReady with empty DiffChanges
   - Calls `transformDelta(delta, currentMetadata)` to get DiffChanges
   - Calls setReady with the result
   - Wraps the whole flow in try/catch: on error, calls setError with the error message

3. **Reset function** `resetDiff(): void`:
   - Sets diffResult=null, diffStatus='idle', diffError=null, notify()
   - Used when navigating away from changelog page (cleanup)
