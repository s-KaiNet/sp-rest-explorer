  <action>
Add a dedicated toolbar row between the page header (title + subtitle) and the summary count cards in ChangelogPage.tsx.

**Range dropdown (left side):**
- Use a native HTML `<select>` element styled with Tailwind (simple, accessible, no dependencies). Style it to match the app's dark mode theme: `rounded-md border border-input bg-background px-3 py-1.5 text-sm`.
- Three options: "Current month" (value: "1"), "Last 3 months" (value: "3"), "Last 6 months" (value: "6")
- Default value: "1" (Current month)
- Store selected value in React state: `const [rangeMonths, setRangeMonths] = useState(1)`

**Filter chips (right side):**
- Three toggle chips: Added, Updated, Removed — all ON by default
- Store as React state: `const [activeFilters, setActiveFilters] = useState<Set<ChangeType>>(new Set(['added', 'updated', 'removed']))`
- Each chip is a `<button>` with conditional styling:
  - Active Added: `bg-green-100 text-green-700 dark:bg-green-900/30 dark:text-green-400 border-green-300 dark:border-green-700` (matches ChangeBadge green)
  - Active Updated: `bg-blue-100 text-blue-700 dark:bg-blue-900/30 dark:text-blue-400 border-blue-300 dark:border-blue-700` (matches ChangeBadge blue)
  - Active Removed: `bg-red-100 text-red-700 dark:bg-red-900/30 dark:text-red-400 border-red-300 dark:border-red-700` (matches ChangeBadge red)
  - Inactive (any): `bg-muted/50 text-muted-foreground border-border` (neutral muted)
- Chip shape: `rounded-full px-3 py-1 text-xs font-medium border cursor-pointer transition-colors`
- Click toggles the chip's ChangeType in the `activeFilters` Set

**Toolbar layout:**
- Flex row: `flex items-center justify-between gap-4 mb-6`
- Make it sticky: `sticky top-14 z-40 bg-background/95 backdrop-blur-sm py-3 -mx-6 px-6 border-b border-border/50` (top-14 because the app header is h-14)
- On narrow screens: Don't worry about responsive stacking — this is desktop-only per project constraints

**Wire range to diff computation:**
- Compute comparison year/month from rangeMonths: Create a helper `getComparisonDate(rangeMonths: number): { year: number; month: number }` that subtracts N months from the current date. For rangeMonths=1, use existing `getDefaultComparisonDate()`. For 3 and 6, compute manually handling year wraparound.
- Update the `useEffect` that calls `computeDiff(year, month)` to depend on the computed comparison date (not the `rangeMonths` directly — compute the year/month first in a `useMemo`)
- When range changes: the existing loading→ready state machine in diff-store handles showing/hiding the spinner automatically

**Update subtitle text:**
- For rangeMonths=1: "Changes in {month label}" (existing behavior)
- For rangeMonths>1: "Changes from {start month label} to {end month label}" where start is N months ago and end is the current comparison month
- Use the existing `getMonthLabel(year, month)` helper

**Wire filters to detail content:**
- Summary count cards: ALWAYS show totals from full diff data regardless of filter state (per CONTEXT.md decision)
- Detail sections (Root Functions + Entities): Filter the arrays before passing to components
  - `diff.functions.filter(f => activeFilters.has(f.changeType))`
  - `diff.entities.filter(e => activeFilters.has(e.changeType))`
- Recalculate `totalChanges` for the detail view from filtered arrays to determine empty state
- When activeFilters is empty (all chips off): Show a "No change types selected" message in place of detail sections:
  ```tsx
  <div className="flex flex-col items-center py-12">
    <Filter className="size-10 text-muted-foreground/40" />
    <p className="text-sm text-muted-foreground mt-3">No change types selected</p>
    <p className="text-xs text-muted-foreground/60 mt-1">Toggle the filter chips above to show changes</p>
  </div>
  ```
  Import `Filter` from lucide-react for the empty state icon.

**Important:** The existing "No changes detected" empty state (ClipboardCheck icon) is for when the diff itself has zero results. The new "No change types selected" empty state is for when filters hide all results. These are two different states — keep both.
