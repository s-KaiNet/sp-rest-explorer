<task type="auto">
  <name>Task 1: Replace dropdown with segmented control, remove stat cards, redesign filter buttons</name>
  <files>app/src/pages/ChangelogPage.tsx</files>
  <action>
Three structural changes in ChangelogPage.tsx:

**1. Replace `<select>` with segmented control:**
- Remove the native `<select>` element (lines ~189-197)
- Create a connected button group (shared borders, no gap between segments) with three segments: "Current month" (default), "Last 3", "Last 6"
- Active segment: filled background using foreground/primary colors (dark fill on light mode, light fill on dark mode) — use `bg-foreground text-background` or similar neutral inversion
- Inactive segments: outlined/ghost style — transparent bg with border, text-muted-foreground
- Use `rounded-l-md` on first, no rounding on middle, `rounded-r-md` on last to create connected appearance
- Keep the segmented control on the LEFT side of the toolbar (same position as current dropdown)
- Wire each segment to `setRangeMonths(1)`, `setRangeMonths(3)`, `setRangeMonths(6)` respectively
- Active state determined by `rangeMonths === value`

**2. Remove the stat cards:**
- Delete the entire "Summary bar" section (the `<div className="flex gap-4">` containing three `flex-1 rounded-xl border` stat cards — approximately lines 249-268)
- The counts (`counts.added`, `counts.updated`, `counts.removed`) are still computed since they're used in filter buttons now

**3. Redesign filter buttons:**
- Increase size to medium: change from `text-xs px-3 py-1` to `text-sm px-4 py-2`
- Change shape from `rounded-full` (pill) to `rounded-md` (rounded rectangle)
- Integrate counts into the label — format as `{Label} ({count})` e.g., "Added (12)". Use the `counts` object already computed
- Update `chipConfig` to not include `activeClass` inline — instead derive active colors from a shared muted palette (will be set in Task 2, for now use placeholder muted colors):
  - Added active: `bg-emerald-100 text-emerald-800 border-emerald-300 dark:bg-green-900/30 dark:text-green-400 dark:border-green-700`
  - Updated active: `bg-sky-100 text-sky-800 border-sky-300 dark:bg-blue-900/30 dark:text-blue-400 dark:border-blue-700`
  - Removed active: `bg-rose-100 text-rose-800 border-rose-300 dark:bg-red-900/30 dark:text-red-400 dark:border-red-700`
- Inactive state: fully desaturated — `bg-muted/50 text-muted-foreground border-border` (already defined as `INACTIVE_CHIP_CLASS`)
- Filter buttons stay on the RIGHT side of the toolbar
