  <action>
    Create the `app/` directory at the repo root. Scaffold a new Vite project using `npm create vite@latest . -- --template react-ts` from within `app/`. This creates the standard React 19 + TypeScript template.

    After scaffolding, modify `vite.config.ts`:
    1. Set `base: '/sp-rest-explorer/'` (for GitHub Pages subdirectory hosting)
    2. Set `build.outDir: '../docs'` with `build.emptyOutDir: true` (output to repo-root docs/)
    3. Add path alias: `resolve.alias: { '@': path.resolve(__dirname, './src') }`
    4. Import `path` from `'path'` at the top

    Update `tsconfig.app.json` to add path alias:
    ```json
    "paths": { "@/*": ["./src/*"] }
    ```
    Also set `"baseUrl": "."` in compilerOptions.

    Ensure TypeScript strict mode is on (Vite template enables it by default — verify).

    Run `npm install` in `app/` to install dependencies.

    Verify the dev server starts: `npm run dev` should serve a page.
    Verify the build works: `npm run build` should produce output in the repo-root `docs/` directory.
  </action>
  <verify>
    Run from `app/` directory:
    1. `npm run build` completes without errors
    2. Verify `../docs/index.html` exists after build
    3. `npx tsc --noEmit` completes without TypeScript errors
  </verify>
  <done>
    `app/` directory exists with a working Vite + React 19 + TS5 project. `npm run build` outputs to repo-root `docs/`. Path alias `@/` is configured in both Vite and TypeScript.
  </done>
