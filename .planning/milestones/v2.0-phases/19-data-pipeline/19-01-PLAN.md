<task type="auto">
  <name>Task 2: Create fetch module with retry, backoff, 429 handling, and timeout</name>
  <files>backend/src/pipeline/fetch-metadata.ts</files>
  <action>
**Create `backend/src/pipeline/fetch-metadata.ts`** exporting a single function `fetchMetadataXml(token: string, spUrl: string): Promise<string>`.

**Constants:**
- `MAX_RETRIES = 3`
- `BASE_DELAY_MS = 2000` (2 seconds)
- `TIMEOUT_MS = 60_000` (60 seconds)

**Main function `fetchMetadataXml`:**
1. Build URL: `${spUrl}/_api/$metadata`
2. Loop `attempt` from 1 to `MAX_RETRIES`:
   a. Create `AbortController` and set 60s timeout via `setTimeout(() => controller.abort(), TIMEOUT_MS)`
   b. Try `axios.get(url, { headers: { Authorization: \`Bearer ${token}\` }, signal: controller.signal, responseType: 'text', transformResponse: [(data: string) => data] })` â€” the `transformResponse` override prevents axios from attempting JSON.parse on the XML response
   c. On success: `clearTimeout(timeoutId)`, return `response.data`
   d. On error: `clearTimeout(timeoutId)` in finally block
      - If `attempt === MAX_RETRIES`: throw the error (exhausted retries)
      - If error is NOT retryable: throw immediately
      - Otherwise: compute delay, await `sleep(delay)`, continue loop

**Helper function `isRetryable(error: unknown): boolean`:**
- Network errors: `axios.isAxiosError(error) && !error.response` â†’ retryable
- Timeout/cancel: `axios.isAxiosError(error) && error.code === 'ERR_CANCELED'` â†’ retryable
- 429 (throttled): `error.response?.status === 429` â†’ retryable
- 5xx server errors: `error.response?.status >= 500` â†’ retryable
- Everything else (4xx except 429): NOT retryable

**Helper function `getRetryDelay(error: unknown, attempt: number): number`:**
- If 429 with `Retry-After` header: parse header value as integer seconds, return `seconds * 1000`
- Otherwise: exponential backoff `BASE_DELAY_MS * Math.pow(2, attempt - 1)` â†’ yields 2000, 4000, 8000ms

**Helper function `sleep(ms: number): Promise<void>`:**
- `return new Promise(resolve => setTimeout(resolve, ms))`

Import axios and AxiosError from 'axios'. Use `.js` extension in relative imports per Node16 module resolution.
