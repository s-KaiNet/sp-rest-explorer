**File 1: `app/src/lib/api-types.ts`** — Create new file.

Define a canonical `ApiType` union type for the entire app:

```typescript
/** The 4 API object types used throughout the icon system */
export type ApiType = 'root' | 'nav' | 'function' | 'entity'
```

This is app-wide scope per Claude's discretion area. Keep it minimal — just the type. No enum, no utility functions (type resolution is for consuming phases).

**File 2: `app/src/components/ui/type-icon.tsx`** — Create new file.

Create a `TypeIcon` component with these specifications (all from locked CONTEXT.md decisions):

**Props:**
- `type: ApiType` — which API type to render
- `size?: 'sm' | 'md' | 'lg'` — defaults to `'sm'`
- `className?: string` — for additional layout classes (margin, positioning), NOT color overrides

**Icon mapping (LOCKED decisions):**
- `root` → Lucide `Box` icon
- `nav` → Lucide `Compass` icon
- `function` → Lucide `Zap` icon
- `entity` → Lucide `Braces` icon
- fallback (unknown) → Lucide `HelpCircle` icon in `text-muted-foreground` (neutral/gray)

**Color mapping — always renders in its type color (LOCKED: no color override prop):**
- `root` → `text-type-root` (the new green Tailwind class)
- `nav` → `text-type-nav` (existing purple)
- `function` → `text-type-fn` (existing blue)
- `entity` → `text-type-entity` (now orange/amber after Task 1)

**Size variants (Claude's discretion on exact values):**
- `sm`: `size-4` (16px) — for sidebar items, list rows, search results
- `md`: `size-5` (20px) — for cards, section headers
- `lg`: `size-9` (36px) — for welcome screens (the big icon boxes)

Use `import { Box, Compass, Zap, Braces, HelpCircle } from 'lucide-react'`.

Implementation approach:
- Use a record/map for icon and color lookups (not a big if/else chain)
- Use `cn()` from `@/lib/utils` to merge className
- Component should be a simple, pure, presentational component — no state, no side effects
- Export as named export: `export function TypeIcon(...)`

Example usage (for documentation in a comment):
```tsx
<TypeIcon type="root" />           // small green Box
<TypeIcon type="function" size="md" /> // medium blue Zap
<TypeIcon type="entity" size="lg" />   // large orange Braces
```
